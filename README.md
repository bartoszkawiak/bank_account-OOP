# Banking System – Python OOP

A small console-based Python project demonstrating core Object-Oriented Programming principles.

## Overview

This project models a simple bank account system where users can:
- Create an account with an owner name and starting balance
- Deposit money with validation
- Withdraw money with validation
- Prevent invalid transactions (negative amounts or insufficient funds)

The focus of this project is clean class design and structured validation.

## Concepts Demonstrated

- Object-Oriented Programming (OOP)
- Encapsulation using private attributes
- Property decorators (@property)
- Input validation
- Exception handling
- Clear method structure
- Separation of concerns (class file + main runner)

## Technologies Used

- Python 3
- OOP principles
